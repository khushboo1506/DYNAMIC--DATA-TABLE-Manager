
# Dynamic Data Table Manager (v3)

## Run locally (Windows)
1. Extract the zip.
2. Open PowerShell in the extracted folder.
3. Install dependencies:
   npm install
4. Start dev server (auto finds a free port 3000-3010):
   npm run dev
5. If it prints 'Starting dev server on port 3001' open http://localhost:3001
6. Clean build cache:
   npm run clean

If you see peer dependency warnings, run:
  npm install --legacy-peer-deps
